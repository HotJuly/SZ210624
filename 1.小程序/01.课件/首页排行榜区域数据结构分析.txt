数据结构:
[
	{
		name:"龙虎榜",
		id:"001",
		list:[
			{
				id:"1001",
				name:"买不到的快乐",
				index:1,
				imgUrl:"xxx.jpg"
			}
		]
	}
]

分析流程:
	1.由于需要在swiper中创建多个swiper-item组件
		所以,此时必须要有一个数组用于遍历

	2.由于每个榜单都有许多不同类型的数据
		所以,数组中必须存放多个榜单对象
		每个榜单都具有自己的名称,歌曲排名信息,唯一标识

	3.由于每个榜单的歌曲排名需要创建多个歌曲卡片
		所以,list属性值必须是一个数组

	4.由于每个歌曲都有许多不同类型的数据
		所以,数组中必须存放多个歌曲对象
		每个歌曲都具有自己的名称,排名,图片,唯一标识
